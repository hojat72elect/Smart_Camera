This is our Android camera app called `Smart Camera` , we're going to gradually add new features to
this project.

Current tech stack used in this app:

* Jetpack Compose (for defining the UI of the app)
* CameraX

Current features of this camera :

<ol>
    <li>Camera live preview.</li>
    <li>Capturing image.</li>
    <li>Switch the camera (forward and backward)</li>
    <li>Saving the captured image to device.</li>
    <li>Turning on and off flash in image capture mode.</li>
    <li>Timer in image capture mode.</li>
</ol>
